Este projeto foi desenvolvido como parte do meu primeiro projeto prático no curso de Frontend da Coti Informática.

A proposta consiste na criação de uma página temática inspirada no filme Michael, utilizando conceitos fundamentais de desenvolvimento web com HTML5 e CSS3. O projeto foi construído com foco em organização visual, responsividade e boas práticas de estilização.

Tecnologias utilizadas
HTML5
CSS3
Flexbox
Media Queries
Funcionalidades e recursos implementados
Estrutura semântica em HTML
Layout centralizado utilizando Flexbox
Imagem de fundo com efeito cover
Cartaz do filme integrado ao layout
Botão estilizado com efeito hover
Página responsiva para dispositivos móveis
Ajustes específicos para impressão com @media print
Integração de link externo para trailer no YouTube
Objetivo do projeto

O objetivo principal foi praticar os fundamentos do Frontend moderno, explorando:

Estruturação de páginas web
Estilização com CSS
Responsividade
Organização visual de componentes
Experiência do usuário

Este projeto representa o início da minha jornada no desenvolvimento Frontend e demonstra a aplicação prática dos primeiros conceitos aprendidos durante o curso.
